安装

在 linux 终端命令行只需要一行指令安装

设置为默认终端

在linux终端输入指令

有时候不是很好用，需要在终端的 preference 中的 command 中设置进入终端的指令即 zsh

配置 oh my zsh

使用 zsh 可以安装一个 oh-my-zsh 用于 zsh 的一些配置和美化，下面两条指令都是安装 oh-my-zsh 的，任选一条安装即可，本质上都是运行 install.sh 文件来进行安装

zsh 主题

这个主题可以从网络上下载，之后在 zsh 的配置文件 .zshrc 中修改对应的主题，需要下载到对应主题名称的 $ZSH_CUSTOM/themes/xxx.zsh-theme 中

zsh 插件

zsh 支持插件，用以提升使用体验，例如 zsh-autosuggestions 能根据历史指令提供指令补全， zsh-syntax-highlighting 插件能提供文本高亮等，另外还可以安装 z 和 extract 等插件，根据自己喜好即可。由于 z 和 extract 是 zsh 自带的插件，只需要下载另外两个插件即可，如下

配置文件

在安装 oh-my-zsh 时，它会选择是否生成配置文件和是否将 zsh 设置为默认终端。也可以不生成自己写，需要注意的是，这个配置文件需要位于 ~/.zshrc

总结

zsh 是一个很强大的终端工具，支持很多常用的工具，例如 ros，ros2 等，在执行 source 时，可以使用对应的 .zsh 文件，就不需要切换终端来进行 source 了，这一点比 fish 做的好一点

上面两条指令都是更新系统设置的环境变量等，当然要想很好的使用 zsh 还需要在安装某些软件配置环境变量时，将对应的指令添加到 ~/.zshrc 文件中
